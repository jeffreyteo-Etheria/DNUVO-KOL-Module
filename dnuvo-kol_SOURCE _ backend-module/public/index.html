Partnership Types & Fit Rules
Two different commission mechanics
LIVE-selling commission and non-LIVE affiliate-shop commission are separate, brand-set rates that each campaign sets for itself — both sit on top of the platform's own standard commission, they don't replace it. Set the actual rates in Pricing · Promo, which also shows the floor-price impact of stacking these.
Hard rules applied to every candidate
• Local audience ≥ 70% (from Setup) — instant exclude below
• Active in last 90 days — dormant accounts excluded
• ER ≥ 3% TikTok / ≥ 2% IG vs tier benchmark
• Profile must fit d.nuvo positioning (clinical ceramide skincare, sensitive skin, K-beauty)
• Cost included where publicly known; otherwise marked "rate on request"
Verification honesty rule
The AI proposes candidates from search knowledge. It cannot click links. Every candidate carries a ⏳ pending link check badge until the backend module's /verify-creator endpoint confirms the profile URL is live and active. No candidate is ever labelled "verified" without that check.
Creator Link Verification Queue (backend)
Paste one creator profile URL per line (TikTok or Instagram). This runs the server-side verifier so you can separate live profiles from dead or blocked links before contract outreach.

Pricing & Promotions Playbook
Review-gate rule
No paid discount depth beyond BAU (flash, LIVE voucher, mega sale) until the primary marketplace listing has 15+ reviews. Below that: gifting/nano seeding and organic content only — this matches the KPI gate in Budget · KPI.
What's per-campaign vs. platform-wide
SKU pricing (SRP/COGS) and KOL commission rates below are set independently per campaign — the d.nuvo SKU list and its 25%/12% KOL rates only pre-fill as a reference when you're working under the DNUVO advertiser; every other advertiser starts with an empty SKU table and 0% KOL commission until you set its own. Platform commission (Shopify/Shopee/Lazada/TikTok Shop) is the one exception — those default the same for every campaign because they're the marketplace's own fee schedule, not something a brand negotiates.
Platform commission rates — edit to match your storefronts
Floor price = (COGS × 4 + S$5) ÷ (1 − platform commission). These are the marketplace's own cut only — LIVE/affiliate KOL commission is separate, see below.
Defaults last checked Jul 2026 — verify in Seller Centre
Shopee: skincare commission runs ~7.6% (non-Mall) to ~15.3% (Mall) plus a 2.18% transaction fee, and a new ~5% technical support fee began rolling out to SG sellers from Feb 2026 — default set to 18% as a conservative blend. TikTok Shop: default platform commission is 8.175% (standard sellers, from the Apr 2026 rate update) — default set to 9%, but exact rate is category-specific. Lazada not independently re-verified this session. Rates vary by category/seller tier — treat these as starting points, not quotes.
Add / import SKUs
Import reads the SRP as the first S$ figure per line from Setup's "Hero SKUs or bundles" field. COGS is not in that field — set it manually per row for an accurate floor.
SKU floor & max safe discount by channel
LIVE & creator-shop affiliate — stacked commission
These two mechanics add a brand-set KOL commission on top of the platform commission above — both are deducted from the same sale, so the real floor is higher than the base channel floor. Each campaign sets its own rate here (defaults to 0% until you set it, except the DNUVO advertiser which pre-fills its own 25%/12% reference rates).
